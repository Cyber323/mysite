Transitioning to Linux

Dual booting

Dual Booting is one of the best ways to get started with Linux. Dual booting works with almost any distro as long as it has grub. Most Ubuntu based distros, like Linux Mint, Feren OS, KDE Neon, and Zorin OS. Other OS's like EndeavourOS will work, because they have grub. If you use Arch, you can install the os-prober and grub packages to dual boot. After you are done setting up the dual boot, you can transfer all your documents from your Windows partition to your Linux partition because Linux can read NTFS! Now, you need to see if there are Linux apps for your applications that you need. Go to the site and download/run the command(s). If there is no Linux version, you can find alternatives. Some alternatives for Photoshop are GIMP, Krita, and Inkscape. Some alternatives to Windows Mail are Thunderbird or Evolution.

Ditching Windows completely

If you are comfortable with Linux and you know how to use it and wish to single-boot Linux, you need to follow these steps. First, check that your Applications work either natively, by Flatpak/Snap/Appimage, or WINE. Then, if everything is compatiable or you've found alternatives, you can fully remove Windows (or MacOS). Back up your important data either on a cloud or through a USB stick (use a second USB because we'll need 2 if you are backing up to a USB.). Then, launch up the installer. Fill out the infornation it asks you to. When you arrive in the disk or partitining section, click 'Remove everything' or 'Erase disk'. Let the install complete. When it says complete, reboot and remove the install USB. Now, login and go to the desktop. If there are more setup things, fill them out. Then, restore your files from the backups. Reinstall all your applications via your preferred method. Now, your whole hard drive has Linux!
