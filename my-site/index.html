Why I Like Linux and Free Software...

I like Linux because you can customize everything, from the init to the desktop environment and looks of the OS. In addition, there's no spyware or keyloggers looks at Windows Previews. To add on, you choose what you want on your PC, not some big company. You choose how you want to install packages and you choose what is right for you. With Linux, you are basically in control of every single thing. I like Free and Open-Source software because you can look into the code (You can also look in the code of Linux distros and kernels because they are open-source). Looking into the code is a good thing because if someone decides to track you or collect data, you would know. In addition, you can distribute Free and Open Source software to others or make a fork of it. (Basically cloning the project and making some changes in the code).
